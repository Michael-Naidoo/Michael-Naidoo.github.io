Interactive Experiences & Technical Demos

A selection of my work spanning game development, procedural generation, and web-based technical challenges. Each project is a deep dive into specific programming concepts and design aesthetics.

Titan Runner: Endless 3D World

An infinite runner built from the ground up to explore procedural level generation and optimization in a mobile-first 3D environment.

**Challenge:** Implement smooth, performant, and truly endless level generation using Unity's Job System for multithreading, ensuring high frame rates even as geometry is recycled and obstacles are placed dynamically.

  • Unity Engine
  • C# / DOTS
  • 3D Modeling (Blender)
  • Mobile Optimization
View Code (GitHub)

Grid Defender: Tactical Tower Defense

A 2D strategy game focusing on complex pathfinding and enemy intelligence on a dynamic grid map, built using the Phaser framework.

**Challenge:** Develop a custom A* pathfinding implementation that recalculates optimal routes in real-time as towers are placed or obstacles change. Focus was placed on creating distinct enemy behaviors (flee, target priority, swarm).

  • Phaser JS
  • JavaScript (ES6)
  • A* Pathfinding
  • Game Physics
Play Demo

AI Sandbox: Algorithm Visualization

An interactive React application to visualize complex algorithms (Dijkstra's, A*) step-by-step, providing clear insights into how they prioritize and process data.

**Challenge:** Ensure the front-end rendering (using canvas or SVG) could smoothly handle real-time updates for thousands of nodes without performance loss, enabling users to interactively modify the grid and watch the changes.

  • React / TypeScript
  • Functional Programming
  • D3.js (for rendering)
  • State Management (Zustand)
Explore Live Demo

Retro Arcade: Space Blaster

A classic, top-down space shooter implemented entirely using vanilla JavaScript and the HTML5 Canvas API to learn low-level game loop architecture.

**Challenge:** Manage sprite sheet animations, collision detection, and score tracking within a single game loop structure. This was a valuable exercise in writing high-performance, dependency-free JavaScript code.

  • HTML5 Canvas
  • Vanilla JavaScript
  • Sprite Animation
  • DOM Manipulation
