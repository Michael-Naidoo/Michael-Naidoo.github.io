Hillclimb Chicken Horse

A 2D sandbox racer made to combine the thrill of a racing platformer with a sandbox based, self built puzzle.

Game Overview & Development Insights

The Vision: Fun, Strategy and Puzzles you dont know you are solving.

Hillclimb Chicken Horse started from the prompt of how can we combine two existing, fun mechanics into a single seamless experience. Our response, combining Hillclimb Racing with Ultimate Chicken Horse to create a puzzle, sandbox, hillclimbing game that is just as fun as its parent games but different enough to have a unique feel.

The game features an active editing system that allows the player to envision with both location and scale waht their level will look like with the new obstacle in place. Future updates will include a custom Wheel collider that is being worked on.

Key Game Features

Level Editing

Maps are populated with obstacles that players will be able to place into the level as they please.

Physics Based Movement

Allows for more variance in each play through.
